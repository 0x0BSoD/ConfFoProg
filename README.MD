### Install

First install kbdd. In Debian:

``` bash
sudo apt install kbdd 
```

Make kbdd autorun. Add __kbdd &__ to __~/.Xsession__ or __~/.xinit__. Then:

``` bash
git clone http://gitlab.servebeer.com/alex/awesomeWM.git ~/.config/awesome
```
